## 🐛 Troubleshooting
**App Won't Install:**
- Enable 'Unknown sources' in Android settings
- Android 5.0+ required
- Minimum 200 MB free storage

**Login Errors:**
- Use exact demo credentials (case-sensitive)
- Check internet connection
- Verify backend is running

**Connection Refused:**
- App connects to: https://yumm-ym2m.onrender.com
- Check Render backend status
- Wait 30 seconds for Render to wake up (free tier)

## 📋 iOS Setup (When Ready)

When you're ready to build the iOS app, follow these steps:

### Step 1: Update `ios/Runner/Info.plist`

The iOS project will be generated by Flutter. When you run:

```bash
flutter create ios
```

Or build for iOS, update `ios/Runner/Info.plist` with:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <!-- ... existing config ... -->

    <!-- Google Sign-In Configuration -->
    <key>CFBundleURLTypes</key>
    <array>
        <dict>
            <key>CFBundleURLName</key>
            <string>com.googleusercontent.apps.946437330680-drp10qt4b720rhdl6h19uruj1pqirsat</string>
            <key>CFBundleURLSchemes</key>
            <array>
                <string>com.googleusercontent.apps.946437330680-drp10qt4b720rhdl6h19uruj1pqirsat</string>
            </array>
        </dict>
    </array>

    <key>REVERSED_CLIENT_ID</key>
    <string>com.google.946437330680-drp10qt4b720rhdl6h19uruj1pqirsat</string>

    <!-- ... rest of config ... -->
</dict>
</plist>
```

### Step 2: Update Podfile

Ensure `ios/Podfile` has minimum platform set:

```ruby
platform :ios, '12.0'
```

### Step 3: Install Pods

```bash
cd ios
pod install
cd ..
```

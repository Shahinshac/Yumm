## 🚀 How It All Works Together

When you click "Run" in Replit:

```
1. .replit file loads
   ↓
2. replit.nix provides Python 3.11 + PostgreSQL
   ↓
3. init_replit.sh script runs automatically:
   - Initialize PostgreSQL ✓
   - Create database ✓
   - Install dependencies ✓
   - Create database schema ✓
   - Generate secure keys ✓
   - Start Flask API ✓
   ↓
4. API available at: https://your-repl.replit.dev
   ↓
5. Frontend (separate on Vercel) connects to backend
```

## 🆘 Troubleshooting

**"Address already in use"?**
- Replit sessions sometimes leave processes. Restart the Replit project.

**"ModuleNotFoundError"?**
- Dependencies didn't install. Delete `.replit` env cache and try again.

**"Connection refused" on database?**
- Wait 5 seconds, PostgreSQL takes time to initialize on first run.

**More help?** See `REPLIT_SETUP.md` troubleshooting section.

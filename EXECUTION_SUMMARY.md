   - Import repository
   - Set frontend path
   - Configure environment

### STEP 2: DEPLOY BACKEND (5-10 minutes)
Follow these steps in Render:
1. Create new Web Service
2. Select bankmanagement repository
3. Add environment variables:
   - `DATABASE_URL` = [from Railway]
   - `JWT_SECRET` = [generate random]
   - `FLASK_ENV` = production
4. Set start command: `gunicorn -w 4 app.run:app`
5. Click Deploy

### STEP 3: DEPLOY FRONTEND (3-5 minutes)
Follow these steps in Vercel:
1. Import bankmanagement repository
2. Set root directory: `frontend`
3. Add environment variables:
   - `REACT_APP_API_URL` = [Render backend URL]
4. Click Deploy

### STEP 4: VERIFY DEPLOYMENT (10-15 minutes)
Test everything:
1. Open frontend URL in browser
2. Login with demo credentials
3. Create/view account
4. Make a transaction
5. Check response times
